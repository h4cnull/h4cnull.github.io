服务器发送邮件溯源测试

h4cnull's blog
2020-02-22

使用gmail账号在服务器上发送邮件，测试溯源。

首先需求将google账号启用“安全性较低的应用的访问权限”。

安装mutt

yum -y install mutt cyrus-sasl-plain

配置mutt，~/.

set from = “xxx@gmail.com“

set realname = “xxx”

set smtp_url = “smtp://username@smtp.gmail.com:587/“

set smtp_pass = “password”

username为邮箱账号 password为密码。

发送邮件

echo “This is a test” | mutt -s “This is a test” xxxxx@qq.com

收件邮件原文，溯源为正确的发送服务器IP。

分析头部https://toolbox.googleapps.com/apps/messageheader/analyzeheader，结果一致。
